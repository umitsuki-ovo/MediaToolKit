## 概要
MediaToolKitは、画像・動画・音声ファイルの圧縮およびフォーマット変換を行うWindows向けデスクトップアプリです。<br>
各種メディアに対して品質・サイズ・形式を柔軟に調整可能。


## ダウンロード & 使い方

1. [ここ](https://github.com/umitsuki-ovo/MediaToolKit/releases)から最新版をダウンロード
2. ZIPを解凍
3. `MediaConverter.exe` を起動


## 動作環境

- OS: Windows11
- CPU: x64
- GPU(なくてもいい)
  -  NVENC : NVIDIA GPU が必要 (GeForce GTX 1000 シリーズ以降)
  -  QSV   : Intel 内蔵グラフィックス (第 6 世代 Core 以降)
  -  AMF   : AMD GPU が必要 (RX 400 シリーズ以降)


## 機能

### 圧縮

拡張子はそのままに、各メディアの品質やサイズを調整して軽量化します。

- 動画
  - 自動最適化（品質調整）
  - 解像度変更（4K → 1080p など）
  - フレームレート（fps）変更
  - コーデック最適化

- 音声
  - 品質優先 / サイズ優先の最適化
  - ビットレート変更
  - サンプリングレート変更

- 画像
  - 非可逆圧縮（品質調整）
  - 可逆圧縮（PNGなど）
  - 解像度変更（プリセット + 任意指定）
  - ファイルサイズ指定圧縮（JPGなど）

---

### 変換

異なるフォーマットへの変換を行います。

- 動画
  - 動画 → 動画（コーデック変換含む）
  - 動画 → 音声（音声抽出）
  - 動画 → 画像（フレーム抽出）

- 音声
  - 音声 → 音声

- 画像
  - 画像 → 画像


## 対応フォーマット

- 動画
  - `.mp4`, `.mov`, `.avi`, `.webm`, `.mkv`, `.m4v`, `.flv`, `.mpg`, `.3gp`

- 音声
  - `.mp3`, `.wav`, `.ogg`, `.aac`, `.m4a`, `.flac`, `.opus`, `.wma`, `.aif`

- 画像
  - `.jpg`, `.png`, `.bmp`, `.gif`, `.tiff`, `.webp`, `.ico`, `.haic`(読み取りのみ), `.svg`(ベータ)


## 詳細設定

- 動画
  - コーデック(mp4): `H.264` / `H.265 (HEVC)` / `MPEG-4`
  - 解像度: 144p ~ 8K
  - フレームレート: 23.98 ~ 240fps

- 音声
  - コーデック(m4a): `AAC` / `ALAC`
  - ビットレート: 96 ~ 320 kbps
  - サンプリングレート: 44.1 ~ 192 kHz
  - ビット深度: 8 ~ 32bit float
